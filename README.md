# express-course

My express-course project.

## Features
- Learn how to build RESTful APIs using Express.js.

## Tech Stack
- Node.js
- Express.js

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/username/express-course.git
   cd express-course
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
